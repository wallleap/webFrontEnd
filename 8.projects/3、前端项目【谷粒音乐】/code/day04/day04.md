	2.导航的布局  橡皮筋效果
		---导航 滑屏区域 与 滑屏元素的布局
			滑屏区域宽度必定占满一个视口
				滑屏区域宽度百分百
			滑屏元素必须被子项撑开
				滑屏元素必须浮动(为了能被子项撑开  禁止子项换行)
				子项统一inline-block
		---无缝滑屏 滑屏区域 与 滑屏元素的布局
			滑屏区域宽度必定占满一个视口
				滑屏区域宽度百分百
			滑屏元素必须被子项撑开
				width:子项个数*100%
				子项：1/子项个数 * 100%
